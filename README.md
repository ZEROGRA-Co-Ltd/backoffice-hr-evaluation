# バックオフィス部門 人事評価制度 WEBページ

株式会社ZEROGRAのバックオフィス部門向け人事評価制度を可視化したWEBページです。

## 📋 現在実装されている機能

### ✅ 完成機能

1. **制度概要セクション**
   - 昇給の目的と背景
   - 昇給の仕組み（年2回、半期+5.0%）
   - 昇給ルールと計算式
   - 昇給制度の特徴
   - 注意事項

2. **昇給制度セクション**
   - 基本設定（現在年俸450万円、次回昇給2026年1月）
   - 計算式の詳細説明（年俸 × 1.05）
   - 昇給のメリット一覧

3. **シミュレーションセクション**
   - 2026年1月から3年間の推移
   - タイムライン表示（7回の昇給ポイント）
   - 詳細テーブル（年俸・月給・増額・増加率）
   - 3年間の成長サマリー
   - 制度のポイント

## 📊 昇給制度の詳細

### 基本設定

- **現在の年俸**: ¥4,500,000（月給¥375,000）
- **次回昇給**: 2026年1月
- **昇給率**: 半期ごとに+5.0%
- **昇給頻度**: 年2回（1月・7月）
- **賞与制度**: 廃止（固定給のみ）

### 3年間のシミュレーション

| 2026年7月 | ¥4,961,250 | ¥413,438 |
| 2027年1月 | ¥5,209,313 | ¥434,110 |

| 2028年1月 | ¥5,743,267 | ¥478,606 |

### 3年間の成長

## 📱 アクセス方法

このWEBページは、以下の方法でアクセス・共有できます：

1. **ブラウザで直接開く**
   - `index.html`をダブルクリックするだけで表示されます
   - インターネット接続不要（スタンドアロン動作）

2. **社内サーバーにアップロード**

3. **URLで共有**
   - Publishタブから公開すると、URLで共有可能になります

## 💡 制度の特徴

### メリット

1. **透明性の高い評価**
   - 自動昇給で評価の不透明さがない
   - 全員平等に報酬が増加

2. **キャリアの見通し**
   - 3年後の報酬を事前に把握可能
   - 長期的なライフプランが立てやすい

3. **シンプルな管理**
   - 賞与評価が不要
   - 人事管理コストの削減

4. **安心感の提供**
   - 勤続年数に応じて確実に昇給
   - 安定した働き方の実現
